# SoundPad Pro - Features

## Virtual Button Support (v2.5.0)

Assign sounds from web URLs to your controller buttons!

### Supported Sources:
- **MyInstants.com** - Sound button pages are automatically detected and extracted
- **Direct Audio URLs** - Any direct link to .mp3, .wav, .ogg, .m4a, etc.

### How to Use:
1. **Shift+Click** any pad to open the URL input dialog
2. Paste a URL (e.g., `https://www.myinstants.com/en/instant/spongebob-fail-11236/`)
3. The app will automatically extract the audio URL
4. Click "Add Sound" to assign it to the button

### Keyboard Shortcuts:
- **Click**: Play sound or assign (if empty)
- **Ctrl+Click**: Open file picker for local audio files
- **Shift+Click**: Open URL input for web sounds
- **Alt+Click**: Assign OBS/LiveSplit actions
- **Right-Click**: Assign OBS/LiveSplit actions (alternative)

### Examples:
- MyInstants sound effects: `https://www.myinstants.com/en/instant/...`
- Direct MP3 links: `https://example.com/sound.mp3`
- Any audio file hosted online

The audio engine supports: MP3, WAV, OGG, M4A, FLAC, WebM, AAC, Opus, WebA
